# GSGFs-blog-frontend

## 目录规划

- `app`: 路由
- `components`: 组件
- `config`: 网站设置
- `hooks`: hooks
- `lib`: 服务端功能函数
- `markdown`: mdx
- `public`: 静态文件
- `scripts`: 辅助脚本
- `server`: 后端数据交换函数
- `style`: 样式表
- `types`: 类型定义
- `utils`: 实用函数

## 如何运行

需要`Node.js`和`pnpm`

1. 安装依赖

   ```bash
   pnpm i
   ```

2. 将`.env.example`复制一份为`.env.local`并填写需要的环境变量

3. 运行

   ```bash
   pnpm run dev
   ```
